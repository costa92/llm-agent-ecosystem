# llm-agent-ecosystem

Top-level umbrella project for the `llm-agent` family.

## Scope

- `llm-agent`
- `llm-agent-rag`
- `llm-agent-otel`
- `llm-agent-providers`
- `llm-agent-customer-support`

## Model

Each subproject remains independently versioned and released.
The ecosystem root only defines shared structure, dependency direction, and navigation.
